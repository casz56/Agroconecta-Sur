# AgroEnergía Conecta Sur — Aplicativo Web

Aplicativo actualizado a partir del proyecto **AgroEnergía Conecta Sur: Plataforma Inteligente para Comunidades Energéticas Agroproductivas, Transición Energética Justa y Uso Productivo de Energía Limpia en el Sur de Colombia**.

## Qué cambió

- Se eliminó el enfoque de marketplace agropecuario tradicional.
- Se reconvirtió el aplicativo hacia transición energética justa y Convocatoria 049.
- Se incorporaron módulos de comunidad energética, generación renovable, consumo, cargas críticas, sensores, IA energética, alertas, reportes, TRL, presupuesto, apropiación social y gobernanza.
- Se mantuvo el logo institucional en `assets/logo-infihuila.png` con ruta relativa compatible con GitHub Pages.

## Archivos

- `index.html`: página institucional del proyecto.
- `plataforma.html`: registro de comunidad energética, cargas críticas y procesos productivos.
- `dashboard.html`: indicadores energéticos, gráficas, alertas y simulador de impacto.
- `reportes.html`: productos CTeI, fases, presupuesto, gobernanza y riesgos.
- `styles.css`: estilos institucionales INFIHUILA.
- `app.js`: interactividad, datos demo, gráficos en canvas y persistencia local.
- `assets/logo-infihuila.png`: logo institucional.

## Publicación en GitHub Pages

1. Sube todos los archivos a la raíz del repositorio.
2. Verifica que la carpeta `assets` quede junto a los HTML.
3. En GitHub: Settings → Pages → Deploy from branch → main / root.
4. Abre la URL generada por GitHub Pages.

No requiere servidor ni dependencias externas.
