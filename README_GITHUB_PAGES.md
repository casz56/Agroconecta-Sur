# AgroEnergía Conecta Sur - App corregida para GitHub Pages

## Instalación correcta

1. Descomprime este ZIP.
2. Sube **todos los archivos de la raíz del ZIP** directamente a tu repositorio de GitHub Pages.
3. Deben quedar en el mismo nivel: `index.html`, `styles.css`, `app.js`, `plataforma.html`, `dashboard.html`, `reportes.html` y la carpeta `assets`.
4. El logo de INFIHUILA también quedó incrustado dentro de los HTML para evitar que se rompa en GitHub Pages si falla la ruta del archivo.
